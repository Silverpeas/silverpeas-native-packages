Silverpeas native packages

Those packages install standalone version of Silverpeas into "/opt/silverpeas" and provide init script, so it can be launched as: 

/etc/init.d/silverpeas start

or: 

service silverpeas start

Since Silverpeas depends on Postgresql the installation will also install Postgresql server on your machine.
But installing Postgresql may require some manual steps: you will find more documentation here.

Debian-based Linux distributions such as Ubuntu

APT

Add the following entry in your /etc/apt/sources.list: 

deb http://www.silverpeas.org/repo/deb binary/

Update your local package index and install Silverpeas: 

sudo apt-get update
sudo apt-get install silverpeas

RPM-based Linux distributions such as RedHat, Fedora, CentOS, openSUSE

YUM

Add repository: 

sudo wget -O /etc/yum.repos.d/silverpeas.repo http://www.silverpeas.org/repo/rpm/silverpeas.repo

Install Silverpeas: 

yum install silverpeas

Zypper

Add repository: 

zypper addrepo http://www.silverpeas.org/repo/rpm silverpeas

Install Silverpeas: 

zypper install silverpeas
